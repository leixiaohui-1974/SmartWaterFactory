## 一、系统定位与设计哲学

### 1.1 背景

本方案面向**水系统控制论（CHS）**理论体系的物理验证需求，聚焦精准加药与精准曝气两大核心控制功能。装备承担双重使命：

- **科研端**：为 HydroClaw 水网大模型提供实验标定数据和控制策略在线验证，支撑 CHS 系列论文的实验证据链；
- **教学端**：成为 AI 时代智慧水务本科生的四年成长伴侣，培养"既懂水、又懂 AI、还会写代码"的新型工程人才。

### 1.2 核心设计原则

| 原则 | 工业科研版 | AquaMind 教育版 |
|------|-----------|----------------|
| 传感精度 | 工业级在线仪器（哈希体系） | 基础传感器 + AI 软测量 |
| 控制精度 | ±0.5%，工业可信 | ±5~10%，教学可接受 |
| 学生可编程度 | 受限（专业工程师操作） | 全开放（从驱动到算法） |
| 造价 | 64 万元（含税） | 3 万元（单套） |
| 目标用户 | 科研人员、研究生 | 大一至大四本科生 |
| HydroClaw 接入 | 全量接入，生产级 | 轻量接入，教学级 |

### 2.6 控制模型与算法

#### 算法一：精准加药——前馈-反馈复合控制

加药系统以水质前馈模型为主导，结合浊度反馈修正，实现动态精准投药。

```python
# 前馈模型（Jar-test 数据驱动）
D_ff = alpha * Q_in * (beta * Turb_in**gamma + delta * pH_correction)

# PID 反馈修正（浊度差值）
e_k     = Turb_target - Turb_out_k
D_fb_k  = Kp * e_k + Ki * sum_e * dt + Kd * delta_e / dt

# 复合投药量（安全上下界约束）
D_total = clip(D_ff + D_fb, D_min, D_max)

# 参数自适应更新（在线学习，每 1h 更新）
# [alpha, beta, gamma, delta] 由 RLS 递推最小二乘法更新
```

#### 算法二：精准曝气——DO 级联 + 氨氮前馈

```python
# 外环：氨氮负荷 → DO 设定点
DO_sp = DO_base + K_NH4 * (NH4_in - NH4_target)

# 内环：DO 闭环 PID → MFC 流量指令
e_DO_k  = DO_sp - DO_meas_k
Q_air_k = Kp_DO * e_DO_k + Ki_DO * integral_e_DO + Kd_DO * deriv_e_DO

# 间歇曝气节能策略（SBR 模式）
T_aer = f(DO_slope, NH4_load)   # 动态曝气时长
T_mix = T_total - T_aer         # 搅拌不曝气阶段

# 能耗优化目标函数
# min J = integral[ w1*(DO-DO_sp)^2 + w2*Q_air^2 ] dt
```

#### 算法三：物理-数据混合水质预测模型（ASM1 + LSTM）

```python
# ASM1 简化核心方程（好氧异养）
# dS_s/dt = -mu_H * (S_s/(K_S+S_s)) * (DO/(K_OH+DO)) * X_H / Y_H
# dX_H/dt =  mu_H * (S_s/(K_S+S_s)) * (DO/(K_OH+DO)) * X_H - b_H * X_H
# dDO/dt  = KLa * (DO_sat - DO) - (1-Y_H)/Y_H * mu_H * ... * X_H

# LSTM 残差修正
# epsilon(t) = y_meas(t) - y_ASM1(t)
# epsilon_pred(t+1) = LSTM( epsilon(t-n:t), u(t-n:t) )
# y_final(t+1) = y_ASM1(t+1) + epsilon_pred(t+1)
```

#### 算法四：模型预测控制 MPC

```python
# MPC 目标函数（Np=10 步，Nc=3 步，Ts=60s）
# min  sum_{k=1}^{Np} [ ||y(t+k|t)-y_ref||_Q^2 + ||delta_u(t+k|t)||_R^2 ]
#
# 状态向量 x = [COD_out, NH4_out, DO_aer, pH, Turb_out, MLSS]
# 控制向量 u = [D_PAC, D_PAM, Q_air_aer, Q_air_anox, D_NaOH]
#
# 约束
# y_min <= y(t+k|t) <= y_max        # 水质约束
# u_min <= u(t+k|t) <= u_max        # 执行器约束
# |delta_u(t+k|t)| <= delta_u_max   # 变化率约束
```

### 2.7 HydroClaw 架构融合

#### 四层架构与实验平台映射

**L4 — 认知 AI 层（云端）**

功能：瀚铎水网大模型推理、MPC 优化求解、强化学习策略优化（PPO）、异常工况识别、多目标决策引擎。
实验平台对应：云端推理层，通过 MQTT/HTTP 接收传感器数据，下发优化控制指令。

**L3 — 物理 AI 层（边缘计算节点）**

功能：ASM1/ASM2d 水质模型、混凝动力学模型、KLa 传质模型、LSTM 残差修正、数字孪生实时同步、参数在线辨识（RLS/EKF）。
实验平台对应：工控机运行实时模型，10 s 更新周期与 HydroClaw 数字孪生同步。

**L2 — 执行控制层（PLC）**

功能：西门子 S7-1200 PLC、DO 级联 PID、加药前馈+反馈、MFC 流量闭环、安全联锁逻辑、OPC-UA 服务器。
实验平台对应：PLC 底层闭环控制，扫描周期 100 ms，OPC-UA 发布过程数据。

**L1 — 感知层（传感器）**

功能：DO/pH/ORP/浊度/MLSS/NH4+/NO3-/COD/流量/液位全量感知，12 类传感器，1 s 采样周期。

#### 通信协议栈

| 接口层级 | 协议 | 数据内容 | 频率/延迟 | 方向 |
|---------|------|---------|---------|------|
| 传感器 → PLC | Modbus RTU / 4-20mA | 原始测量值 | 100 ms～60 s | 上行 |
| PLC → 边缘节点 | OPC-UA | 过程值 + 状态字 | 1 s | 上行 |
| 边缘节点 → 云端 | MQTT over TLS | 特征量 + 模型状态 | 10 s | 双向 |
| HydroClaw → 边缘 | HTTP REST / gRPC | MPC 优化指令包 | < 500 ms | 下行 |
| 数据持久化 | InfluxDB + PostgreSQL | 全量时序 + 事件 | 1 s/10 s | 存储 |

#### 算法二：自适应间歇曝气控制（Year 2~3）

```python
class AdaptiveAeration:
    def __init__(self, DO_target=2.0):
        self.DO_sp = DO_target
        self.pid = PID(Kp=0.8, Ki=0.05, Kd=0.1)

    def update(self, DO_meas, NH4_load):
        # 外环：DO 设定点随氨氮负荷动态调整
        self.DO_sp = 1.5 + 0.8 * (NH4_load / NH4_ref)

        # 内环：PID → MFC 指令
        Q_air = self.pid.compute(self.DO_sp, DO_meas)

        # 节能判断：DO 充足时切换搅拌不曝气（节能 20~35%）
        if DO_meas > self.DO_sp + 0.3:
            return {'mode': 'mix_only', 'Q_air': 0}
        else:
            return {'mode': 'aerate', 'Q_air': max(0.5, min(Q_air, 5.0))}
```

### 6.1 混凝投药前馈模型——完整实现

#### 6.1.1 数学基础

混凝剂最优投量 $D^*$ 由以下多元非线性函数描述：

$$D^* = \alpha \cdot Q_{in} \cdot \left( \beta \cdot \text{Turb}_{in}^{\gamma} + \delta \cdot f_{pH} + \varepsilon \cdot f_T \right)$$

其中：

- $f_{pH} = \exp\left(-\frac{(pH - pH_{opt})^2}{2\sigma_{pH}^2}\right)$，最优 pH 修正高斯函数，$pH_{opt} = 7.2$，$\sigma_{pH} = 0.8$
- $f_T = 1 + \theta \cdot (T - T_{ref})$，温度修正线性项，$T_{ref} = 20°C$，$\theta = 0.02$

参数向量 $\boldsymbol{\theta} = [\alpha, \beta, \gamma, \delta, \varepsilon]$ 由 RLS 递推最小二乘法在线更新。

#### 6.1.2 RLS 在线参数辨识

```python
class OnlineRLS:
    """
    递推最小二乘法（带遗忘因子）
    用于在线更新混凝前馈模型参数
    """
    def __init__(self, n_params: int = 5, forgetting: float = 0.98):
        self.theta = np.zeros(n_params)          # 参数向量初值
        self.P = np.eye(n_params) * 1000.0       # 协方差矩阵（大初值→快速收敛）
        self.lam = forgetting                     # 遗忘因子 λ（0.95~0.99）

    def update(self, phi: np.ndarray, y_meas: float) -> np.ndarray:
        """
        phi : 回归向量（特征向量），shape (n_params,)
        y_meas : 实测投量或出水浊度（标量）
        返回  : 更新后的参数向量
        """
        # 预测误差
        y_pred = phi @ self.theta
        error  = y_meas - y_pred

        # 增益向量
        K = self.P @ phi / (self.lam + phi @ self.P @ phi)

        # 参数更新
        self.theta += K * error

        # 协方差矩阵更新
        self.P = (self.P - np.outer(K, phi @ self.P)) / self.lam

        return self.theta

    def build_phi(self, Q_in, Turb_in, pH, T) -> np.ndarray:
        """构建回归向量 φ"""
        f_pH = np.exp(-((pH - 7.2)**2) / (2 * 0.8**2))
        f_T  = 1 + 0.02 * (T - 20.0)
        return np.array([
            Q_in,
            Q_in * Turb_in**0.5,    # γ≈0.5 初始猜测
            Q_in * Turb_in,
            Q_in * f_pH,
            Q_in * f_T
        ])
```

```python
import numpy as np
from dataclasses import dataclass, field
from typing import Optional
import time

@dataclass
class DosingConfig:
    """加药控制器参数配置"""
    # PID 参数
    Kp: float = 0.15          # 比例增益（mL/min 每 NTU）
    Ki: float = 0.02          # 积分增益
    Kd: float = 0.05          # 微分增益
    # 安全约束
    D_min: float = 0.1        # 最小投药量 mL/min
    D_max: float = 15.0       # 最大投药量 mL/min
    anti_windup: float = 5.0  # 积分抗饱和限幅
    # 目标
    Turb_target: float = 1.0  # 出水浊度目标 NTU
    # 更新周期
    dt: float = 60.0          # 控制周期 s


class PrecisionDosingController:
    """
    精准加药复合控制器
    前馈：Jar-test 数据驱动模型（RLS 在线更新）
    反馈：PID（出水浊度闭环）
    """
    def __init__(self, cfg: DosingConfig):
        self.cfg = cfg
        self.rls = OnlineRLS(n_params=5, forgetting=0.98)
        # PID 内部状态
        self._integral = 0.0
        self._prev_error = 0.0
        self._prev_time = None
        # 历史记录（用于自适应）
        self._history = []

    def compute(
        self,
        Q_in:       float,   # 进水流量 L/h
        Turb_in:    float,   # 进水浊度 NTU
        Turb_out:   float,   # 出水浊度 NTU（反馈）
        pH:         float,   # 水温 pH
        T:          float,   # 水温 °C
        lab_dose:   Optional[float] = None,  # 人工化验投量（用于在线更新）
    ) -> dict:
        """
        计算综合投药量
        返回: {
            'D_total': 总投量 mL/min,
            'D_ff': 前馈分量,
            'D_fb': 反馈分量,
            'error': 浊度偏差,
            'params': 当前 RLS 参数
        }
        """
        now = time.time()
        dt  = self.cfg.dt if self._prev_time is None else (now - self._prev_time)
        self._prev_time = now

        # ── 前馈计算 ──────────────────────────────────────
        phi  = self.rls.build_phi(Q_in, Turb_in, pH, T)
        D_ff = float(phi @ self.rls.theta)
        D_ff = max(self.cfg.D_min, D_ff)  # 非负约束

        # 在线更新（若有人工化验值）
        if lab_dose is not None:
            self.rls.update(phi, lab_dose)

        # ── PID 反馈 ──────────────────────────────────────
        error = self.cfg.Turb_target - Turb_out

        # 积分项（含抗饱和）
        self._integral += error * dt
        self._integral  = np.clip(
            self._integral,
            -self.cfg.anti_windup,
             self.cfg.anti_windup
        )

        # 微分项（低通滤波防噪声）
        d_error = (error - self._prev_error) / max(dt, 1e-6)
        self._prev_error = error

        D_fb = (self.cfg.Kp * error
              + self.cfg.Ki * self._integral
              + self.cfg.Kd * d_error)

        # ── 综合投量 ──────────────────────────────────────
        D_total = np.clip(D_ff + D_fb, self.cfg.D_min, self.cfg.D_max)

        record = {
            'D_total': D_total,
            'D_ff':    D_ff,
            'D_fb':    D_fb,
            'error':   error,
            'params':  self.rls.theta.copy(),
            'timestamp': now
        }
        self._history.append(record)
        return record
```

### 6.2 DO 级联控制器——完整实现

#### 6.2.1 KLa 传质模型辨识

在不同气量 $Q_{air}$ 下，DO 从 $DO_0$ 恢复至饱和值的过程满足：

$$\frac{d(DO)}{dt} = K_La(Q_{air}) \cdot (DO_{sat} - DO) - OUR$$

其中 $K_La$ 与气量关系拟合为幂函数：

$$K_La = a \cdot Q_{air}^b \quad (a, b \text{ 由实验辨识})$$

$DO_{sat}$ 受温度影响：$DO_{sat}(T) = 14.62 - 0.3898 T + 0.006969 T^2 - 0.00005896 T^3$

```python
class KLaIdentifier:
    """
    KLa 在线辨识器
    利用 DO 响应曲线斜率估计当前 KLa
    方法：指数拟合法（每次曝气开始时触发）
    """
    def __init__(self):
        self.a = 0.08    # KLa = a * Q_air^b 初始参数
        self.b = 0.75
        self._buffer_do  = []
        self._buffer_t   = []
        self._active     = False

    def start_identification(self):
        """曝气开始时调用，清空缓存"""
        self._buffer_do.clear()
        self._buffer_t.clear()
        self._active = True

    def push(self, t: float, do: float, do_sat: float):
        """每个采样周期推入数据"""
        if not self._active:
            return
        # 转化为线性化形式：ln(DO_sat - DO) = ln(DO_sat - DO_0) - KLa*t
        deficit = do_sat - do
        if deficit > 0.05:
            self._buffer_t.append(t)
            self._buffer_do.append(np.log(deficit))

    def estimate(self, Q_air: float) -> Optional[float]:
        """
        数据足够时估计 KLa，并更新幂函数参数
        返回当前估计的 KLa 值
        """
        if len(self._buffer_t) < 10:
            return None
        t  = np.array(self._buffer_t) - self._buffer_t[0]
        y  = np.array(self._buffer_do)
        # 线性回归：y = c0 - KLa * t
        A      = np.vstack([np.ones_like(t), t]).T
        coeffs = np.linalg.lstsq(A, y, rcond=None)[0]
        KLa_est = -coeffs[1]   # 斜率取反

        # 更新幂函数参数（对数空间线性化）
        if KLa_est > 0 and Q_air > 0:
            # 记录 (ln Q_air, ln KLa) 点，滚动窗口最小二乘
            self._update_power_law(np.log(Q_air), np.log(KLa_est))

        self._active = False
        return KLa_est

    def predict_KLa(self, Q_air: float) -> float:
        """根据当前 a, b 预测 KLa"""
        return self.a * (Q_air ** self.b)

    def _update_power_law(self, ln_Q, ln_KLa):
        """滑动窗口线性回归更新 a, b（略）"""
        pass

    @staticmethod
    def do_sat(T: float) -> float:
        """饱和溶解氧（mg/L），T 单位°C"""
        return 14.62 - 0.3898*T + 0.006969*T**2 - 0.00005896*T**3
```

```python
@dataclass
class AerationConfig:
    # 外环（氨氮 → DO 设定点）
    DO_base:    float = 2.0    # 基准 DO mg/L
    K_NH4:      float = 0.6    # 氨氮前馈增益
    NH4_ref:    float = 20.0   # 氨氮参考负荷 mg/L
    DO_min:     float = 0.5    # DO 设定点下限
    DO_max:     float = 5.0    # DO 设定点上限
    # 内环（DO → 气量）
    Kp_DO:      float = 1.2
    Ki_DO:      float = 0.08
    Kd_DO:      float = 0.20
    # 执行器约束
    Q_min:      float = 0.5    # 最小气量 L/min
    Q_max:      float = 20.0   # 最大气量 L/min
    dQ_max:     float = 2.0    # 最大变化率 L/min/step
    # 节能参数
    DO_hysteresis: float = 0.3  # 迟滞带，防频繁切换
    dt:            float = 30.0 # 控制周期 s


class CascadedAerationController:
    """
    DO 级联控制器
    外环：氨氮/DO 前馈 → DO 设定点
    内环：DO PID → MFC 气量指令
    附加：间歇曝气节能逻辑
    """
    def __init__(self, cfg: AerationConfig):
        self.cfg   = cfg
        self.kla_id = KLaIdentifier()
        # 内环 PID 状态
        self._integral   = 0.0
        self._prev_err   = 0.0
        self._prev_Q     = cfg.Q_min
        self._mode       = 'aerate'   # 'aerate' / 'mix_only'
        self._mode_timer = 0.0
        # 统计
        self.energy_saved = 0.0   # 节能累计（L·min 气量等效）

    # ── 外环：动态 DO 设定点 ──────────────────────────────
    def compute_DO_setpoint(
        self,
        NH4_in:   float,    # 进水氨氮 mg/L
        NH4_out:  float,    # 出水氨氮 mg/L（可选）
        MLSS:     float,    # 污泥浓度 g/L
        T:        float,    # 水温 °C
    ) -> float:
        """
        氨氮前馈 + 负荷修正
        高氨氮负荷时提高 DO，节能时降低 DO
        """
        load_factor = NH4_in / max(self.cfg.NH4_ref, 1e-3)
        # 温度修正：温度低时微生物活性下降，需更多 DO
        T_correction = 1.0 + 0.03 * max(0, 20.0 - T)
        DO_sp = self.cfg.DO_base * load_factor * T_correction
        return float(np.clip(DO_sp, self.cfg.DO_min, self.cfg.DO_max))

    # ── 内环：DO PID ──────────────────────────────────────
    def compute_air_flow(
        self,
        DO_sp:   float,    # DO 设定点
        DO_meas: float,    # 实测 DO
        T:       float,    # 水温
    ) -> dict:
        err = DO_sp - DO_meas

        # 积分（抗饱和）
        self._integral = np.clip(
            self._integral + err * self.cfg.dt,
            -self.cfg.Q_max / self.cfg.Ki_DO,
             self.cfg.Q_max / self.cfg.Ki_DO
        )

        # 微分
        d_err = (err - self._prev_err) / self.cfg.dt
        self._prev_err = err

        Q_pid = (self.cfg.Kp_DO * err
               + self.cfg.Ki_DO * self._integral
               + self.cfg.Kd_DO * d_err)

        # KLa 前馈补偿（若已辨识）
        KLa = self.kla_id.predict_KLa(self._prev_Q)
        DO_sat = KLaIdentifier.do_sat(T)
        # Q_ff: 维持当前 DO 所需的最小气量（前馈）
        OUR_est = KLa * (DO_sat - DO_meas)  # 估算耗氧速率
        Q_base = max(0, self._prev_Q - OUR_est * 0.1)

        Q_raw = Q_base + Q_pid

        # 变化率限幅
        dQ = np.clip(Q_raw - self._prev_Q, -self.cfg.dQ_max, self.cfg.dQ_max)
        Q_cmd = np.clip(self._prev_Q + dQ, self.cfg.Q_min, self.cfg.Q_max)
        self._prev_Q = Q_cmd

        # ── 间歇曝气节能判断 ──────────────────────────────
        mode = self._intermittent_logic(DO_meas, DO_sp)
        if mode == 'mix_only':
            self.energy_saved += Q_cmd * self.cfg.dt / 60.0
            Q_cmd = 0.0

        return {
            'Q_air':   Q_cmd,
            'DO_sp':   DO_sp,
            'mode':    mode,
            'err_DO':  err,
            'KLa_est': KLa,
        }

    def _intermittent_logic(self, DO_meas: float, DO_sp: float) -> str:
        """
        迟滞间歇曝气逻辑
        DO > DO_sp + hysteresis → 切换搅拌（省气）
        DO < DO_sp - hysteresis → 恢复曝气
        """
        if self._mode == 'aerate':
            if DO_meas > DO_sp + self.cfg.DO_hysteresis:
                self._mode = 'mix_only'
        else:
            if DO_meas < DO_sp - self.cfg.DO_hysteresis:
                self._mode = 'aerate'
        return self._mode

    def step(self, obs: dict) -> dict:
        """
        统一控制步
        obs 字段：DO, NH4_in, NH4_out, MLSS, T
        """
        DO_sp = self.compute_DO_setpoint(
            obs['NH4_in'], obs.get('NH4_out', 0),
            obs['MLSS'], obs['T']
        )
        return self.compute_air_flow(DO_sp, obs['DO'], obs['T'])
```

```python
import numpy as np
from scipy.integrate import solve_ivp
from dataclasses import dataclass

@dataclass
class ASM1Params:
    """ASM1 简化模型参数（可在线辨识更新）"""
    mu_H_max: float = 6.0     # 1/d
    K_S:      float = 20.0    # mgCOD/L
    K_OH:     float = 0.2     # mgO2/L
    b_H:      float = 0.62    # 1/d
    Y_H:      float = 0.67
    mu_A_max: float = 0.8     # 1/d
    K_NH:     float = 1.0     # mgN/L
    K_OA:     float = 0.4     # mgO2/L
    b_A:      float = 0.15    # 1/d
    Y_A:      float = 0.24
    theta_mu: float = 1.072   # 增长速率温度修正
    T_ref:    float = 20.0    # 参考温度 °C


def asm1_rhs(t, state, p: ASM1Params, inputs: dict) -> list:
    """
    ASM1 右端项（6 状态）
    state = [S_s, S_NH, X_H, X_A, DO, X_inert]
      S_s     : 易降解底物 mgCOD/L
      S_NH    : 氨氮 mgN/L
      X_H     : 活性异养菌 mgCOD/L
      X_A     : 活性自养菌 mgCOD/L
      DO      : 溶解氧 mgO2/L
      X_inert : 惰性悬浮固体 mgVSS/L
    inputs = {Q_in, Q_out, V, DO_sat, KLa, Q_air,
              S_s_in, S_NH_in, X_in, T}
    """
    S_s, S_NH, X_H, X_A, DO, X_in = state

    # 温度修正
    tc = p.theta_mu ** (inputs['T'] - p.T_ref)

    # ── 过程速率 ──────────────────────────────────────────
    # 过程 1：好氧异养增长
    mu_H  = (p.mu_H_max * tc
             * (S_s   / (p.K_S   + S_s))
             * (DO    / (p.K_OH  + DO)))
    rho_1 = mu_H * X_H

    # 过程 2：异养内源呼吸
    rho_2 = p.b_H * tc * X_H

    # 过程 3：好氧自养增长（硝化）
    mu_A  = (p.mu_A_max * tc
             * (S_NH / (p.K_NH + S_NH))
             * (DO   / (p.K_OA + DO)))
    rho_3 = mu_A * X_A

    # 过程 4：自养内源呼吸
    rho_4 = p.b_A * tc * X_A

    V   = inputs['V']     # 反应池有效容积 L
    Q   = inputs['Q_in']  # 流量 L/h → 转为 L/min
    HRT = V / max(Q, 1e-6)  # h

    # ── 质量平衡 ──────────────────────────────────────────
    # 各状态导数（单位：mg/L/h，ODE 内部用 1/d 则乘 24）
    conv = 24.0  # 1/d → 1/h

    dS_s  = (inputs['S_s_in'] - S_s) / HRT - rho_1 / p.Y_H  / conv
    dS_NH = (inputs['S_NH_in'] - S_NH) / HRT \
            - (1/p.Y_A) * rho_3 / conv \
            + 0.07 * (rho_2 + rho_4) / conv   # 内源释放氮（简化）
    dX_H  = (inputs['X_in'] - X_H) / HRT + (rho_1 - rho_2) / conv
    dX_A  = -X_A / HRT + (rho_3 - rho_4) / conv
    dX_in = (inputs['X_in'] * 0.2 - X_in) / HRT  # 惰性组分（简化）

    # DO 方程（KLa 由气量辨识提供）
    KLa   = inputs['KLa']          # 1/h
    OUR   = ((1 - p.Y_H) / p.Y_H * rho_1
           + (4.57 - p.Y_A) / p.Y_A * rho_3
           + rho_2 + rho_4) / conv  # mgO2/L/h
    dDO   = KLa * (inputs['DO_sat'] - DO) - OUR \
            + (inputs['DO_in'] - DO) / HRT

    return [dS_s, dS_NH, dX_H, dX_A, dDO, dX_in]


class ASM1DigitalTwin:
    """
    ASM1 数字孪生（含 EKF 实时参数更新）
    每 10 s 与真实传感器同步一次状态
    """
    def __init__(self, params: ASM1Params, V: float = 300.0):
        self.p   = params
        self.V   = V
        self.state = np.array([100.0, 20.0, 1500.0, 200.0, 2.0, 500.0])
        self._t  = 0.0

    def predict(self, inputs: dict, dt_h: float = 10/3600) -> np.ndarray:
        """向前积分 dt_h 小时"""
        inputs['V'] = self.V
        sol = solve_ivp(
            asm1_rhs, [0, dt_h],
            self.state,
            args=(self.p, inputs),
            method='RK45',
            max_step=dt_h / 5,
            rtol=1e-4, atol=1e-6
        )
        self.state = sol.y[:, -1]
        self._t   += dt_h
        return self.state

    def assimilate(self, DO_meas: float, NH4_meas: float = None):
        """
        状态同化：用传感器实测值修正数字孪生状态
        简单线性卡尔曼更新（教学版）
        """
        K_DO  = 0.6   # DO 观测增益
        K_NH4 = 0.4   # 氨氮观测增益
        self.state[4] += K_DO * (DO_meas - self.state[4])
        if NH4_meas is not None:
            self.state[1] += K_NH4 * (NH4_meas - self.state[1])

    @property
    def COD_eff(self) -> float:
        """出水 COD 估算（mgCOD/L）"""
        return self.state[0] + self.state[2] * 0.1 + self.state[5] * 0.05

    @property
    def NH4_eff(self) -> float:
        return max(0, self.state[1])
```

### 6.4 LSTM 软测量模型——完整架构

#### 6.4.1 模型架构

```python
import torch
import torch.nn as nn
from torch.utils.data import Dataset, DataLoader

class WaterQualityLSTM(nn.Module):
    """
    水质软测量 LSTM 模型
    输入：多步传感器序列 + 控制量历史
    输出：COD、NH4+、TP 同步预测（多任务）
    """
    def __init__(
        self,
        input_dim:   int = 10,    # DO, pH, EC, NTU, T, ORP, Q_air, D_PAC, D_NaOH, hour_sin
        seq_len:     int = 24,    # 历史步数（24 步 × 5min = 2h）
        hidden_dim:  int = 128,
        num_layers:  int = 2,
        dropout:     float = 0.2,
        output_dim:  int = 3,     # [COD, NH4, TP]
    ):
        super().__init__()
        self.seq_len    = seq_len
        self.hidden_dim = hidden_dim

        # 特征嵌入层（线性投影 + LayerNorm）
        self.input_proj = nn.Sequential(
            nn.Linear(input_dim, hidden_dim),
            nn.LayerNorm(hidden_dim),
            nn.GELU(),
        )

        # 双向 LSTM
        self.lstm = nn.LSTM(
            hidden_dim, hidden_dim,
            num_layers  = num_layers,
            batch_first = True,
            dropout     = dropout if num_layers > 1 else 0,
            bidirectional = True,
        )

        # 注意力机制（时间步加权，找到最重要的时刻）
        self.attn = nn.Sequential(
            nn.Linear(hidden_dim * 2, 64),
            nn.Tanh(),
            nn.Linear(64, 1),
        )

        # 多任务输出头（各指标独立头，共享主干）
        self.head_COD = self._make_head(hidden_dim * 2)
        self.head_NH4 = self._make_head(hidden_dim * 2)
        self.head_TP  = self._make_head(hidden_dim * 2)

    def _make_head(self, in_dim: int) -> nn.Module:
        return nn.Sequential(
            nn.Linear(in_dim, 64),
            nn.GELU(),
            nn.Dropout(0.1),
            nn.Linear(64, 1),
            nn.Softplus(),    # 保证输出非负
        )

    def forward(self, x: torch.Tensor) -> dict:
        """
        x: (batch, seq_len, input_dim)
        返回: {'COD': (batch,1), 'NH4': (batch,1), 'TP': (batch,1)}
        """
        # 特征投影
        h = self.input_proj(x)          # (B, T, hidden)

        # LSTM
        lstm_out, _ = self.lstm(h)       # (B, T, hidden*2)

        # 注意力聚合
        attn_w = torch.softmax(self.attn(lstm_out), dim=1)  # (B, T, 1)
        context = (attn_w * lstm_out).sum(dim=1)             # (B, hidden*2)

        return {
            'COD': self.head_COD(context).squeeze(-1),
            'NH4': self.head_NH4(context).squeeze(-1),
            'TP':  self.head_TP(context).squeeze(-1),
        }


class WaterQualityDataset(Dataset):
    """时序数据集，支持滑动窗口切片"""
    def __init__(self, df, seq_len=24, target_cols=('COD_lab','NH4_lab','TP_lab')):
        self.seq_len = seq_len
        feat_cols = ['DO','pH','EC','NTU','T','ORP','Q_air','D_PAC','D_NaOH','hour_sin']
        self.X = torch.tensor(df[feat_cols].values, dtype=torch.float32)
        self.y = torch.tensor(df[list(target_cols)].values, dtype=torch.float32)

    def __len__(self):
        return len(self.X) - self.seq_len

    def __getitem__(self, idx):
        return (
            self.X[idx : idx + self.seq_len],    # 序列特征
            self.y[idx + self.seq_len]            # 当前时刻标签
        )


def train_one_epoch(model, loader, optimizer, device='cpu'):
    """
    多任务加权损失训练（COD 权重更高，因量程大）
    """
    model.train()
    weights = {'COD': 1.0, 'NH4': 5.0, 'TP': 10.0}  # 归一化补偿
    total_loss = 0.0

    for X_batch, y_batch in loader:
        X_batch, y_batch = X_batch.to(device), y_batch.to(device)
        optimizer.zero_grad()
        pred = model(X_batch)

        loss = (weights['COD'] * nn.functional.mse_loss(pred['COD'], y_batch[:,0])
              + weights['NH4'] * nn.functional.mse_loss(pred['NH4'], y_batch[:,1])
              + weights['TP']  * nn.functional.mse_loss(pred['TP'],  y_batch[:,2]))
        loss.backward()
        nn.utils.clip_grad_norm_(model.parameters(), max_norm=1.0)
        optimizer.step()
        total_loss += loss.item()

    return total_loss / len(loader)
```

### 6.5 MPC 控制器——do-mpc 完整实现

```python
import do_mpc
from casadi import *

def build_aquamind_mpc(
    V_aer: float = 200.0,   # 好氧区容积 L
    dt_s:  float = 60.0,    # 采样周期 s
    Np:    int   = 12,       # 预测域步数
    Nc:    int   = 3,        # 控制域步数
) -> tuple:
    """
    构建 AquaMind MPC 控制器
    状态：[S_s, S_NH, DO, MLSS, Turb_out]  (5 states)
    控制：[Q_air, D_PAC, D_NaOH]           (3 controls)
    返回：(model, mpc, simulator)
    """
    model = do_mpc.model.Model('continuous')

    # ── 状态变量 ──────────────────────────────────────────
    S_s      = model.set_variable('_x', 'S_s',      (1,1))  # 底物 mgCOD/L
    S_NH     = model.set_variable('_x', 'S_NH',     (1,1))  # 氨氮 mgN/L
    DO       = model.set_variable('_x', 'DO',       (1,1))  # DO mg/L
    MLSS     = model.set_variable('_x', 'MLSS',     (1,1))  # 污泥 mg/L
    Turb_out = model.set_variable('_x', 'Turb_out', (1,1))  # 浊度 NTU

    # ── 控制变量 ──────────────────────────────────────────
    Q_air  = model.set_variable('_u', 'Q_air',  (1,1))  # 曝气量 L/min
    D_PAC  = model.set_variable('_u', 'D_PAC',  (1,1))  # PAC 投量 mL/min
    D_NaOH = model.set_variable('_u', 'D_NaOH', (1,1))  # 碱液投量 mL/min

    # ── 干扰/参数 ──────────────────────────────────────────
    S_s_in  = model.set_variable('_p', 'S_s_in',  (1,1))  # 进水底物
    S_NH_in = model.set_variable('_p', 'S_NH_in', (1,1))  # 进水氨氮
    T_water = model.set_variable('_p', 'T_water', (1,1))  # 水温
    KLa     = model.set_variable('_p', 'KLa',     (1,1))  # 传质系数（辨识值）
    DO_sat  = model.set_variable('_p', 'DO_sat',  (1,1))  # 饱和 DO

    # ── 常数 ──────────────────────────────────────────────
    mu_H = 6.0 / 24.0   # 1/h
    K_S  = 20.0; Y_H = 0.67; K_OH = 0.2
    mu_A = 0.8 / 24.0
    K_NH = 1.0;  Y_A = 0.24; K_OA = 0.4
    HRT  = V_aer / 30.0   # h，假设流量 30 L/h

    # ── ODE 右端项（CasADi 符号计算）────────────────────────
    rho_1 = mu_H * (S_s/(K_S+S_s)) * (DO/(K_OH+DO)) * MLSS / Y_H * 0.45
    rho_3 = mu_A * (S_NH/(K_NH+S_NH)) * (DO/(K_OA+DO)) * MLSS * 0.05

    model.set_rhs('S_s',      (S_s_in - S_s)/HRT - rho_1)
    model.set_rhs('S_NH',     (S_NH_in - S_NH)/HRT - rho_3/Y_A)
    model.set_rhs('DO',       KLa*(DO_sat-DO) - (1-Y_H)/Y_H*rho_1 - 4.57*rho_3
                              + (0.5 - DO)/HRT)
    model.set_rhs('MLSS',     -MLSS/HRT + 0.45*rho_1 + 0.12*rho_3)
    model.set_rhs('Turb_out', (D_PAC * 0.02 * MLSS - Turb_out * 0.5) * 0.01)
    model.setup()

    # ── MPC 控制器 ────────────────────────────────────────
    mpc = do_mpc.controller.MPC(model)
    setup_mpc = {
        'n_horizon': Np,
        'n_robust':  0,
        't_step':    dt_s / 3600.0,   # CasADi 使用小时为单位
        'store_full_solution': True,
    }
    mpc.set_param(**setup_mpc)

    # 目标函数（终端项 + 阶段代价）
    # 权重矩阵 Q（状态偏差）和 R（控制量惩罚）
    mterm = (S_NH - 5.0)**2 * 100 + (Turb_out - 1.0)**2 * 20
    lterm = (mterm
           + (DO - 2.0)**2 * 50
           + Q_air**2 * 0.5        # 曝气能耗
           + D_PAC**2  * 1.0       # 药剂成本
           + D_NaOH**2 * 2.0)

    mpc.set_objective(mterm=mterm, lterm=lterm)
    mpc.set_rterm(Q_air=0.1, D_PAC=0.2, D_NaOH=0.3)  # 控制增量惩罚

    # 约束
    mpc.bounds['lower', '_x', 'DO']       = 0.1
    mpc.bounds['upper', '_x', 'DO']       = 8.0
    mpc.bounds['lower', '_x', 'S_NH']     = 0.0
    mpc.bounds['lower', '_u', 'Q_air']    = 0.5
    mpc.bounds['upper', '_u', 'Q_air']    = 20.0
    mpc.bounds['lower', '_u', 'D_PAC']    = 0.1
    mpc.bounds['upper', '_u', 'D_PAC']    = 15.0
    mpc.bounds['lower', '_u', 'D_NaOH']   = 0.0
    mpc.bounds['upper', '_u', 'D_NaOH']   = 5.0

    mpc.setup()
    return model, mpc
```
