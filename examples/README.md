# Examples and Development Guide

Welcome to the development guide for the Water Plant Process Controller project. This collection of examples will walk you through the framework, from basic usage to advanced customization.

Each folder contains a self-contained example with a `README.md` file that explains the concepts and the code in detail.

## The Tutorials

### 1. Beginner
-   **[01 - Basic Simulation](./01_basic_simulation/README.md)**: Learn how to set up and run a simple, default simulation. This is the best place to start.

### 2. Intermediate
-   **[02 - PID Controller Tuning](./02_tuning_pid_controller/README.md)**: See the dramatic effect that PID tuning has on controller performance.
-   **[03 - Comparing Controller Performance](./03_comparing_controllers/README.md)**: Compare the sophisticated `PIDController` to the simple `OnOffController` to understand the trade-offs.

### 3. Advanced
-   **[04 - Advanced Simulation Features](./04_advanced_simulation_features/README.md)**: Learn how to enable and configure more realistic simulation features like time delays and non-linear effects.
-   **[05 - Extending the Simulator](./05_extending_the_simulator/README.md)**: A step-by-step guide on how to modify the simulator and add your own physical behaviors to the model.
